# پروژه DigiPay UI 🚀

این پروژه یک پیاده‌سازی مدرن از رابط کاربری DigiPay است که با استفاده از **HTML**، **Tailwind CSS** و **JavaScript** توسعه داده شده است.  
این وب‌سایت به‌صورت کاملاً **ریسپانسیو** طراحی شده و روی تمامی دستگاه‌ها (موبایل، تبلت و دسکتاپ) به‌درستی نمایش داده می‌شود.  
برای ساخت اسلایدرها نیز از کتابخانه **Swiper.js** استفاده شده است.

## ✨ توضیحات پروژه

این پروژه با هدف تمرین و تقویت مهارت‌های فرانت‌اند طراحی شده است. تمرکز اصلی در این پروژه روی موارد زیر بوده:

- طراحی رابط کاربری مدرن و مشابه نمونه واقعی
- پیاده‌سازی اصول طراحی ریسپانسیو
- استفاده از Swiper برای ایجاد اسلایدرهای حرفه‌ای
- کدنویسی تمیز و قابل توسعه با JavaScript
- بهبود تجربه کاربری (UX)
